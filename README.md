# Instruções de Uso

## Criação de novos formulários

Para criar um novo formulário siga os seguintes passos:
- Duplique esse modelo
- Insira o link gerado no google apps scripts na página index.html, no local onde está escrito "LINK_AQUI"

## Encerrar um formulário

Para encerrar o formulário, siga as seguintes instruções:
- Insira a tag de inicio de comentário (<!--) antes de "<iframe" 
- Insira a tag de encerramento de comentário (-->) após "</script>"
- Remova as mesmas tags de início e fim de comentário abaixo de "Sessão para encerrar o formulário"
- Assim aparecerá uma mensagem na tela indicando o fim das inscrições.
